# Midnight SaaS Design System

Este documento define a linguagem visual do ecossistema Kanux Pro.

## Princípios Visuais
- **Foco em Contraste:** Uso de tons escuros profundos (#131313) com destaques em azul vibrante (#3b82f6).
- **Profundidade:** Utilização de glassmorphism e elevação via cores de container (`surface-container`).
- **Clareza UX:** Tipografia Inter para máxima legibilidade em dispositivos móveis.
